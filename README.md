# 🎓 Internship Management Portal


A robust, role-based platform designed to bridge the gap between students, faculty, corporate managers, and university administrators. This system streamlines internship tracking, documentation, and academic credit integration.


# ⚙️ Installation & Setup

1. Clone the repository

``` 
https://github.com/ShriyaAl/WebDevProject-Internship-Experential-Learning-Tracker.git
```
2. Install Dependencies
```
npm install
```
3. Start Development Server
```
npm run dev
```
